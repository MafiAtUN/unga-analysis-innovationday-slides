# UNGA Analysis Platform - Innovation Day Presentation

**AI-Powered Intelligence for UN Office of the Special Adviser on Africa**

## 🎯 Presentation Overview

This is a comprehensive 19-slide presentation showcasing the UNGA Analysis Platform, an AI-powered system for analyzing United Nations General Assembly speeches.

### Key Highlights

- **20,000+ speeches** from 200 countries (1946-2025)
- **AI-powered analysis** using RAG (Retrieval-Augmented Generation)
- **Real-time insights** and trend analysis
- **Interactive features** including semantic search and visualizations
- **Production-ready** solution with Azure cloud deployment

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **AI/ML:** Azure OpenAI API, Sentence Transformers
- **Database:** DuckDB (Vector DB with 384-dim embeddings)
- **Visualization:** Interactive charts and graphs
- **Cloud:** Azure App Service, Azure Storage

## 💡 Key Features Demonstrated

- **Semantic Search:** Find speeches by meaning, not keywords
- **Cross-Year Analysis:** Track evolution across 80 years
- **SDG Tracking:** Monitor all 17 Sustainable Development Goals
- **AI Chat:** Ask questions about UNGA speeches
- **Interactive Visualizations:** Trend analysis and comparisons
- **Cost Calculator:** Dynamic cost analysis based on users
